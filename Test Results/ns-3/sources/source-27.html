41 /**
42  * Base class representing a tick unit.  This determines the spacing of the
43  * tick marks on an axis.
44  * <P>
45  * This class (and any subclasses) should be immutable, the reason being that
46  * ORDERED collections of tick units are maintained and if one instance can be
47  * changed, it may destroy the order of the collection that it belongs to.
48  * In addition, if the implementations are immutable, they can belong to
49  * multiple collections.
50  *
51  * @see ValueAxis
52  */
53 public abstract class TickUnit implements Comparable, Serializable {
